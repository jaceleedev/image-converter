# 릴리즈 노트

## 2.5.0 — 대화형 이미지 변환 마무리 (2026-05-01)

이번 릴리즈는 웹 개발에 자주 쓰는 실제 이미지 파일 변환 흐름을 대화형 모드 중심으로 다듬은 배포입니다.

### 주요 변경

- 인자 없이 실행하면 대화형 모드로 바로 시작합니다.
- 단일 파일 변환에서 출력 확장자가 선택한 포맷과 다르면 변환 전에 차단합니다.
- 단일 파일 대화형 모드의 기본 저장 경로를 입력 파일과 같은 폴더의 자연스러운 확장자 변경 경로로 제안합니다.
- 기존 출력 파일은 덮어쓰지 않습니다. 단일 변환은 중단하고, 일괄 변환은 해당 파일만 건너뜁니다.
- 대화형 모드에서 최대 가로 크기 기반 리사이즈를 지원합니다.
- JPEG 출력 시 투명 영역을 흰색, 검정, 직접 입력한 `#RRGGBB` 배경색 위에 합성할 수 있습니다.
- 출력 포맷 선택지, 품질 프리셋, 안내 문구를 웹 이미지 변환 흐름에 맞게 정리했습니다.

### 지원 범위

- 입력: PNG, JPG/JPEG, WebP, AVIF(8-bit), TIFF/TIF, BMP, ICO
- 출력: PNG, JPG/JPEG, WebP, AVIF
- PDF와 HEIC 입력은 이번 릴리즈 범위에 포함하지 않았습니다. PDF는 페이지 렌더링 정책이 필요한 문서 변환 기능이고, HEIC는 `libheif` 계열 시스템 의존성이 필요해 별도 큰 작업으로 다룹니다.

### 검증

- `./scripts/check.sh` 로 포맷팅, Clippy, 테스트를 함께 통과시키는 것을 기준으로 합니다.
